# Agent Workflow (RPI Loop)

All tasks MUST follow this loop.

## 1. Research

* Understand existing system
* Identify files, patterns, constraints
* Do NOT write code yet

## 2. Plan

Output:

* Goal
* Files to modify/create
* Step-by-step plan
* Data flow
* Risks

## 3. Implement

* Follow plan exactly
* No extra changes
* Keep scope minimal

## 4. Verify

* Type correctness
* Edge cases handled
* No regressions
* Matches requirements

## Rule

Never skip directly to implementation.
