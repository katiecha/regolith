# Verification

## Types

### 1. Evidence Check

Label claims:

* verified
* inference
* unknown

### 2. Format Check

Ensure:

* required sections exist
* correct structure
* no missing parts

## When Required

* high-risk tasks
* long outputs
* system design
* data logic

## Rule

Never present assumptions as facts
